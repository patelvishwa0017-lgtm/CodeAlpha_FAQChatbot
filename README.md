# CodeAlpha FAQ Chatbot

## Project Overview

This project is an AI-powered FAQ Chatbot developed using Natural Language Processing (NLP). The chatbot answers user questions by finding the most similar question in the FAQ dataset and returning the corresponding answer.

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* Natural Language Processing (NLP)
* Google Colab

## Features

* FAQ Question Matching
* NLP-based Similarity Search
* Interactive Chatbot
* Chat History Storage
* Custom FAQ Dataset

## Dataset

The project uses a custom FAQ dataset containing 50+ questions and answers related to Artificial Intelligence, Machine Learning, Programming, and Technology.

* CodeAlpha_FAQChatbot.ipynb
